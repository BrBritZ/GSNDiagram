Safety-Critical Systems: Open Assessment 2016-17

Using GSN to Integrate Software Safety and Cyber Security

by Teerawat Teeravatcharoenchai ID:2273032

This tools use for creating GSN(Goal Structuring Notation) Diagram for safety and security case in complex software. Users can drag and drop main element of GSN on the left side to create GSN diagram on the right side. They can also create a link between each element and adjust text inside the element.
